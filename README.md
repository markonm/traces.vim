# traces.vim

## Overview
This plugin will highlight patterns and ranges for EX commands.

It also provides live preview for the following EX commands:
```
:substite
:smagic
:snomagic
```

**Note:** live preview is currently Vim-only feature.

## Requirements
Vim v8.0.1206+ or Neovim v0.2.3+
